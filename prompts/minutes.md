# 議事録生成スキル

以下の「トランスクリプト（会話の文字起こし）」と「議題（会議資料）」をもとに、Markdown形式で簡潔に整理された議事録を作成してください。

## 出力形式

```markdown
## 1. 決定事項
- [決定した内容]

## 2. 次のアクション
- [アクションアイテムと担当者]

## 3. 議論の概要
- [議論された主なポイント]
```

## ルール

- 入力にない事実、氏名、日付、金額を推測しない。
- 決定事項と検討中の内容を明確に分ける。
- 担当者や期限が不明なアクションは「要確認」と記載する。
